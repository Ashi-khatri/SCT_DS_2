# SCT_DS_2 – Titanic Data Cleaning & EDA

## Objective
To perform data cleaning and exploratory data analysis on the Titanic dataset.

## Dataset
Titanic passenger dataset with demographic and survival details.

## Data Cleaning
- Identified missing values
- Filled missing Age values using mean

## EDA Performed
- Survival analysis
- Gender vs survival
- Passenger class vs survival
- Age distribution
- Fare vs survival

## Key Insights
- Females had higher survival rates
- First class passengers survived more
- Higher fare passengers had better survival chances

## Tools Used
Python, Pandas, Matplotlib, Seaborn
